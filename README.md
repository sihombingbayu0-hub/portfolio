# Portfolio Personal

Website portfolio statis yang siap dipublikasikan melalui GitHub Pages.

## Personalisasi

Edit `index.html`, lalu ganti bagian berikut:

- `Nama Anda` dan inisial `NA`
- `emailanda@example.com`
- Semua `USERNAME` dengan username GitHub/LinkedIn Anda
- Semua `NAMA-REPOSITORY` dengan nama repository proyek Anda
- Deskripsi profil, keahlian, statistik, dan proyek

## Menjalankan di komputer

Buka `index.html` langsung di browser, atau gunakan ekstensi Live Server di Visual Studio Code.

## Publikasi ke GitHub Pages

1. Buat repository baru di GitHub, misalnya `portfolio`.
2. Upload `index.html`, `styles.css`, dan `script.js` ke repository tersebut.
3. Buka **Settings → Pages**.
4. Pada **Build and deployment**, pilih **Deploy from a branch**.
5. Pilih branch `main`, folder `/ (root)`, lalu tekan **Save**.
6. Tunggu beberapa saat. Website akan tersedia di `https://USERNAME.github.io/portfolio/`.
